# فروشگاه کمبو گیمینگ 🎮

یک فروشگاه آنلاین استاتیک با تم دارک گیمینگ برای فروش کمبوهای خوشمزه. این پروژه کاملاً استاتیک است و با HTML، CSS و JavaScript خالص ساخته شده است. می‌تواند بدون نیاز به سرور یا بک‌اند روی GitHub Pages اجرا شود.

## 📁 ساختار پروژه

```
frontend/
│
├── index.html          # صفحه اصلی فروشگاه
├── styles.css          # استایل‌های دارک گیمینگ
├── app.js              # منطق فروشگاه (محصولات، سبد، سفارش)
├── products.json       # فایل داده محصولات
├── README.md          # این فایل
│
├── admin/             # پنل مدیریت
│   ├── index.html     # صفحه اصلی پنل
│   ├── login.html     # صفحه ورود
│   ├── admin.css      # استایل‌های پنل
│   └── admin.js       # منطق پنل مدیریت
│
└── assets/            # پوشه تصاویر
    ├── combo1.jpg     # تصویر محصول 1 (placeholder)
    ├── combo2.jpg     # تصویر محصول 2 (placeholder)
    └── ...
```

## 🚀 نحوه اجرای پروژه

### روش 1: اجرای محلی

1. به پوشه `frontend` بروید:
   ```bash
   cd frontend
   ```

2. فایل `index.html` را با دابل کلیک باز کنید

3. یا از یک سرور محلی استفاده کنید:
   ```bash
   # با Python
   python -m http.server 8000
   
   # با Node.js (http-server)
   npx http-server
   ```

4. مرورگر را باز کنید و به آدرس `http://localhost:8000` بروید

### روش 2: اجرا روی GitHub Pages

1. پروژه را در یک repository روی GitHub آپلود کنید
2. به Settings → Pages بروید
3. Source را روی `main` (یا `master`) و `/frontend` تنظیم کنید
4. Save کنید
5. بعد از چند دقیقه، سایت شما در آدرس زیر در دسترس خواهد بود:
   ```
   https://YOUR_USERNAME.github.io/REPOSITORY_NAME
   ```

## 🔧 تنظیمات مهم

### 1. تغییر شماره کارت پرداخت

فایل `app.js` را باز کنید و خطوط 15-18 را ویرایش کنید:

```javascript
const PAYMENT_CONFIG = {
    cardNumber: '6037-9981-9893-7616',  // شماره کارت خود را وارد کنید
    cardName: 'ابولفضل دوست گل'            // نام صاحب حساب را وارد کنید
};
```

### 2. تنظیم Formspree برای دریافت سفارشات

1. به سایت [Formspree.io](https://formspree.io) بروید
2. یک حساب کاربری بسازید (رایگان)
3. یک فرم جدید ایجاد کنید
4. آدرس endpoint را کپی کنید (مثل: `https://formspree.io/f/abc123xyz`)
5. فایل `app.js` را باز کنید
6. خط ۴۳۰ را پیدا کنید:
   ```javascript
   const formspreeUrl = 'https://formspree.io/f/XXXXX';
   ```
7. `XXXXX` را با آدرس endpoint خود جایگزین کنید:
   ```javascript
   const formspreeUrl = 'https://formspree.io/f/abc123xyz';
   ```

### 3. تغییر رمز ورود پنل مدیریت

فایل `admin/login.html` را باز کنید و خط ۳۱ را ویرایش کنید:

```javascript
const ADMIN_PASSWORD = 'admin123';  // رمز خود را وارد کنید
```

## 📝 نحوه کار با پنل مدیریت

### ورود به پنل

1. از صفحه اصلی فروشگاه، روی لینک "🔐 پنل مدیریت" کلیک کنید
2. یا مستقیماً به `admin/login.html` بروید
3. رمز مدیریت را وارد کنید (پیش‌فرض: `admin123`)
4. پس از ورود موفق، به پنل مدیریت هدایت می‌شوید

### مدیریت محصولات

1. در پنل مدیریت، روی "📦 مدیریت محصولات" کلیک کنید
2. برای افزودن محصول جدید:
   - روی دکمه "+ افزودن محصول جدید" کلیک کنید
   - فرم را پر کنید:
     - نام محصول
     - توضیح کوتاه
     - توضیح کامل
     - قیمت (به تومان)
     - لینک تصویر (مثل: `assets/combo1.jpg`)
     - محتویات کمبو (هر آیتم در یک خط)
   - روی "ذخیره" کلیک کنید
3. برای ویرایش محصول:
   - روی دکمه "✏️ ویرایش" در کارت محصول کلیک کنید
   - اطلاعات را تغییر دهید
   - روی "ذخیره" کلیک کنید
4. برای حذف محصول:
   - روی دکمه "🗑️ حذف" کلیک کنید
   - در modal تأیید، روی "بله، حذف کن" کلیک کنید

**نکته مهم:** تمام تغییرات در localStorage ذخیره می‌شوند و در فروشگاه نمایش داده می‌شوند.

### مدیریت سفارش‌ها

1. در پنل مدیریت، روی "📋 مدیریت سفارش‌ها" کلیک کنید
2. لیست تمام سفارش‌های ثبت شده نمایش داده می‌شود
3. برای تغییر وضعیت سفارش:
   - از dropdown "وضعیت" استفاده کنید
   - وضعیت را به "در انتظار" یا "تکمیل شده" تغییر دهید
4. برای حذف سفارش:
   - روی دکمه "🗑️ حذف" کلیک کنید
   - در modal تأیید، تأیید کنید

## 📝 نحوه افزودن یا تغییر محصولات (دستی)

اگر می‌خواهید مستقیماً فایل `products.json` را ویرایش کنید:

1. فایل `products.json` را باز کنید
2. ساختار هر محصول به این صورت است:

```json
{
  "id": 1,
  "title": "نام محصول",
  "shortDescription": "توضیح کوتاه",
  "description": "توضیح کامل محصول",
  "price": 85000,
  "image": "assets/combo1.jpg",
  "items": [
    "آیتم 1",
    "آیتم 2",
    "آیتم 3"
  ]
}
```

3. محصول جدید را به آرایه اضافه کنید یا محصول موجود را ویرایش کنید
4. برای هر محصول جدید، یک تصویر در پوشه `assets/` قرار دهید
5. فایل را ذخیره کنید و صفحه را رفرش کنید

**نکته:** اگر از پنل مدیریت استفاده کنید، تغییرات در localStorage ذخیره می‌شوند و اولویت دارند.

## 🎨 سفارشی‌سازی

### تغییر رنگ‌های تم

فایل `styles.css` را باز کنید و متغیرهای CSS در بخش `:root` را تغییر دهید:

```css
:root {
    --neon-blue: #00d4ff;      /* رنگ آبی نئونی */
    --neon-purple: #b026ff;    /* رنگ بنفش نئونی */
    --neon-pink: #ff00ff;      /* رنگ صورتی نئونی */
    --neon-green: #00ff88;     /* رنگ سبز نئونی */
    /* ... */
}
```

### تغییر فونت‌ها

فایل `styles.css` را باز کنید و خط 5 را ویرایش کنید:

```css
@import url('https://fonts.googleapis.com/css2?family=Orbitron:wght@400;700;900&family=Vazir:wght@300;400;500;700&display=swap');
```

می‌توانید فونت‌های دیگر را از Google Fonts انتخاب کنید.

### تغییر نام فروشگاه

فایل `index.html` را باز کنید و خط 16 را ویرایش کنید:

```html
<h1 class="logo-text">
    <span class="neon-blue">⚡</span>
    فروشگاه کمبو
    <span class="neon-purple">🎮</span>
</h1>
```
